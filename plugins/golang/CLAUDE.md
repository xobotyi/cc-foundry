# golang Plugin

Go language discipline plugin. Teaches Go-specific conventions, idioms, and toolchain practices.

## Skills

| Skill | Purpose |
|-------|---------|
| `golang` | Go language conventions, idioms, error handling, concurrency, and toolchain |
| `templ` | Templ (a-h/templ) HTML templating syntax, component patterns, and Go integration |

## Plugin Scope

This plugin covers Go language specifics and Go-specific tooling like templ.
Language-agnostic coding practices are provided by the `the-coder` plugin.
Platform-specific concerns (backend, CLI) are provided by their respective
platform plugins.
