# javascript

JavaScript and TypeScript language discipline plugin for Claude Code.
Provides skills for JS/TS conventions, patterns, and ecosystem practices.

## Installation

```
/plugin marketplace add xobotyi/cc-foundry
/plugin install javascript
```

## What's Inside

### Skills

- **typescript** — JavaScript and TypeScript conventions, patterns,
  type system practices, and ecosystem tooling.

## Related Plugins

- **the-coder** — Language-agnostic coding discipline (discovery,
  planning, verification)
- **frontend** — Frontend platform concerns (CSS, accessibility)
- **backend** — Backend platform concerns (observability, API design)
- **cli** — CLI platform concerns
