# javascript Plugin

JavaScript and TypeScript language discipline plugin. Teaches JS/TS-specific
conventions, patterns, and ecosystem practices.

## Skills

| Skill | Purpose |
|-------|---------|
| `typescript` | JavaScript and TypeScript conventions, patterns, and type system practices |

## Plugin Scope

This plugin covers JavaScript and TypeScript language specifics.
Language-agnostic coding practices are provided by the `the-coder` plugin.
Platform-specific concerns (frontend, backend, CLI) are provided by their
respective platform plugins.
