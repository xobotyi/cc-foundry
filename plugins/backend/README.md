# backend

Backend platform discipline plugin for Claude Code. Provides skills for
service-oriented development practices.

## Installation

```
/plugin marketplace add xobotyi/cc-foundry
/plugin install backend
```

## What's Inside

### Skills

- **logging** — Logging conventions, structured logging, log levels,
  and best practices for service observability.

## Related Plugins

- **golang** — Go language discipline
- **javascript** — JavaScript and TypeScript language discipline
- **the-coder** — Language-agnostic coding discipline (discovery,
  planning, verification)
