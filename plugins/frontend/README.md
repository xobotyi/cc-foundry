# frontend

Frontend platform discipline plugin for Claude Code. Provides skills for
browser-specific development practices.

## Installation

```
/plugin marketplace add xobotyi/cc-foundry
/plugin install frontend
```

## What's Inside

### Skills

- **css** — CSS conventions, methodologies, layout patterns, and
  best practices.

## Related Plugins

- **javascript** — JavaScript and TypeScript language discipline
- **the-coder** — Language-agnostic coding discipline (discovery,
  planning, verification)
