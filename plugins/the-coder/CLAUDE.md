# the-coder Plugin

Output styles and skills for general software engineering work.

## Output Styles

| Style | Purpose |
|-------|---------|
| `software-engineer` | Implementation-focused persona with engineering judgment |

## Skills

| Skill | Purpose |
|-------|---------|
| `coding` | Universal coding discipline: discover before assuming, verify before coding |

## Plugin Scope

This plugin covers language-agnostic development practices.
Language-specific skills (Go, TypeScript, React, etc.) belong in
dedicated discipline plugins.
